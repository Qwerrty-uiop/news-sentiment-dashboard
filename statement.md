\## Problem Statement

In a fast-paced world, it's difficult to quickly gauge the overall public sentiment or media portrayal of a company, product, or event. This project aims to solve this by providing a tool that aggregates live news and automatically analyzes its sentiment, offering an instant, data-driven overview of the topic's current "vibe."



\## Scope of the project

The application is a fully functional web tool. Its scope includes taking user input for a topic, fetching data from an external news API, performing sentiment analysis on the data using a robust NLP model, and displaying the results in a clear, visual, and interactive dashboard.



\## Target users

This tool is designed for anyone interested in media monitoring, from stock market analysts and PR professionals to curious individuals wanting to understand how a topic is being covered in the news.



\## High-level features

\- Real-time, topic-based news aggregation.

\- Nuanced sentiment classification (Positive, Negative, Neutral).

\- Web-based graphical user interface (GUI).

\- Aggregate sentiment visualization via a bar chart.
